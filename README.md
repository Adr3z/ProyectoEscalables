## Descripción
BakePOS es un sistema de punto de venta diseñado para cubrir necesidades operativas esenciales como:
- Registro de Ventas
- Control de inventario
- Administración de Productos
- Consulta de Reportes


### Estado del proyecto
**Primer Avance:** Martes 24 de marzo de 2026

Implementación de la mayoría de vistas del sistema.

Diseño de interfaz de usuario.

### Tecnologías
- Angular
- Tailwind CSS
